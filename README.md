# SkyRelics

Spigot plugin for configurable relic items, custom item abilities, and the Herobrine ritual logic.

Commands:

- `/skyrelics give <player> <item_id> [amount]`
- `/skyrelics giveall <item_id> [amount]`
- `/skyrelics list [page]`
- `/skyrelics info <item_id>`
- `/skyrelics inspect`
- `/skyrelics validate`
- `/skyrelics abilities [page]`
- `/skyrelics cooldown clear <player> [key|all]`
- `/skyrelics reload`

Alias:

- `/sr`

- `docs/abilities.md`
